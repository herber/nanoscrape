# Nanosnap

Small service for getting screenshots of websites. 

## Install

```
$ npm install nanosnap
```

## Usage

### Start server

```
nanosnap
```

### Get a screenshot

```
https://<instance-url>/?url=https://google.com
```

## CLI

```
$ npm install --global nanosnap
```

```
$ nanosnap

  Usage
    nanosnap

  Options
    --port The port you want nanosnap to listen to [Default: 3000]

  Examples
    $ nanosnap --port 80
```
